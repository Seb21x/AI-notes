Run app.py, wait for program to compile, first compilation might take some time. Then go to this link: http://127.0.0.1:5000
Requirements:
- Flask==3.1.1
- Flask-SQLAlchemy==3.1.1
- transformers==4.53.2
- torch==2.7.1
- requests==2.32.4
